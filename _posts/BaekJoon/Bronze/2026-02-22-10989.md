## 2. 문제 분석
#### 🐾 생각 흐름 (1차시도-메모리 초과) 
되게 쉬워서 저는 브론즈 이제 다 통달한 줄 알았습니다. <br>
그런데 아니나 다를까 메모리 초과가 떴습니다. 하하.....

```python
import sys

def input() :
    return sys.stdin.readline().strip()

n = int(input())
numbers = []

for i in range(n) :
    numbers.append(int(input()))

numbers.sort()

for i in numbers : 
    print(i)
```
초간단 코드니까 설명은 생략하겠습니다. 


#### 🐾 생각 흐름 (2차시도-맞았습니다) 
1. 시간을 빠르게 하기 위해 sys.stdin.readline() 을 썼습니다. 
2. 그리고 어떤 숫자의 등장횟수를 담는 리스트 cnt를 만들었습니다. <br> N은 10,000보다 작거나 같은 자연수라는 제한이 있기 때문에 10001이라는 제한을 두었습니다.
3. 그 뒤, 숫자를 n개 입력 받으며 개수를 셉니다. <br> 이게 첫번째 루프이고 숫자를 저장하지 않고 몇 번 나왔는지만 기록하는 역할입니다. 
4. 0부터 10000까지 순서대로 확인하면서 해당 숫자가 존재하면 등장횟수만큼 반복해서 출력하도록합니다. (두 번째 루프)

```python
import sys

def input() :
    return sys.stdin.readline().strip()

n = int(input())
cnt = [0] * 10001

for _ in range(n) : 
    cnt[int(input())] += 1

for i in range(10001) :
    if cnt[i] != 0 :
        for j in range(cnt[i]) :
            print(i)
```
