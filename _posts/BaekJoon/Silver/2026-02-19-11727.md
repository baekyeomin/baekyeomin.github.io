## 2. 문제 분석
#### 🐾 생각 흐름
저... 1차원 배열 생성하는 방법을 까먹었었습니다... 하하.....<br>
맨첨에 [[0] *(n+1)]해놓고 이게왜안돼?? 이러고 있었어요 ㅎ.ㅋ..ㅎㅋ...<br>
검색하면서 풀었답니다..ㅋㅋ... <br>

```python
n = int(input())
dp = [0]*(n+1)
dp[1] = 1

if n >=2 :
    dp[2] = 3

for i in range(3, n+1) :
    dp[i] = dp[i-2]*2 + dp[i-1]

print(dp[n]%10007)
    
```
