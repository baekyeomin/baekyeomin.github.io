## 2. 문제 분석
#### 🐾 생각 흐름

```python

def input() :
    return sys.stdin.readline()

n, k = map(int,input().split())
nums = [i for i in range(1,n+1)]
answer = []
x = 0

while len(answer) != n :
    x = (x+k-1)%len(nums)
    
    answer.append(nums.pop(x))
    
print("<", end="")
for i in range(len(answer)):
    if i < len(answer)-1:
        print(answer[i], end=", ")
    elif i >= len(answer)-1:
        print(answer[i], end="")
print(">")
```

[기억할 사항]
