## 2. 문제 분석
#### 🐾 생각 흐름 (1차시도-맞았습니다) 
1. 수의 길이 n 입력 받기

2. 처음 dp = [1,<span style="color:green">1</span>,1,1,1,1,1,1,1,1]
   - 길이가 1인 (= 한 자릿수인) 수들의 마지막 자릿수가 0~9일때 각각 경우에서 오르막 수의 개수
  
3. 누적합을 구하도록함 
   - 예를들어 세자릿수 중 마지막 수가 4인 수의 오르막 수 개수는 004, 014, 024, 034, 044, 114, 124, 134, 144, 224, 234, 244, 334, 344, 444로 총 15개이다. <br> <span style="background-color: #fff3cd">끝자리가 d인 오르막 수는 이전 자리 끝이 0~d인 모든 경우에서 만들어지는 것 </span>을 알 수 있다. 그래서 누적합을 쓸 수 있다. 
  
4. _ = 2일때 [<span style="color:green">1</span>,<span style="background-color: #ceffcdff">2</span>,3,4,5,6,7,8,9,10]
    - 점화식 : dp[d] = dp[d] + dp[d-1] (녹색부분 참고)
5. _ = 3 일 때 [1,3,6,10,15,21,28,36,45,55]
6. 이런식으로 n까지 반복해서 구한뒤, 합을 구해 10007로 나눈 나머지를 답으로 출력하게 한다. 


```python
n = int(input())

dp = [1] *10

for _ in range(2, n + 1):
    for d in range(1,10):
        dp[d] = dp[d] + dp[d-1]

print(sum(dp) % 10007)
    
```
