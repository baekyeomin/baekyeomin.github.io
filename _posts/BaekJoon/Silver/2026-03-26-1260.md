## 2. 문제 분석
저는 그래프 문제 풀기를 3학년이 되도록 회피해오던 학생이었기에.. <br> 오랜만에 본 블로그의 자료구조글까지 올려가면서 다시 그래프 공부 후 문제를 풀었습니다.  

### 🐾 생각 흐름 (1차시도-컴파일에러) 
```python
import sys

def input() :
    return sys.stdin.readline().split()

n,m,v = map(int,input())

graph = [[0]*(n+1) for _ in range(n+1)]

for _ in range(m) :
    a,b = map(int,input())
    graph[a][b] = graph[b][a] = 1
    
visited = []

def dfs(v) :
    global visited
    visited[v] = 1
    print(v,end= ' ')
    for next in range(1,n+1) :
        if not visited[next] and graph[v][next] :
            dfs(next)
            
def bfs() :
    global q, visited
    while q :
        cur = q.pop(0)
        print(cur,end = ' ')
        for next in range(1,n+1) :
            if not visited[next] and graph[cur][next]:
            visited[next] = 1
            q.append(next)
        
dfs(v)
print()
visited = [0]*(n+1)
q = [v]
visited[v] =1
bfs()
```
visited = []로 초기화를 하면 visited[v] = 1에서 에러가 발생하더랍니다.. <br>
visited = [0] * (n + 1)로 초기화를 했어야했던..<br>
그리고 next로 썼던 변수명도 생각해보니까 파이썬 내장함수 이름이랑 같아서,,, 바꾸기로했어용 <br> <br>

### 🐾 생각 흐름 (2차시도 - 맞았습니다)
```python
import sys

def input():
    return sys.stdin.readline().split()

n, m, v = map(int, input())

graph = [[0]*(n+1) for _ in range(n+1)]

for _ in range(m):
    a, b = map(int, input())
    graph[a][b] = graph[b][a] = 1

visited = [0]*(n+1)

def dfs(v):
    visited[v] = 1
    print(v, end=' ')
    for i in range(1, n+1):
        if not visited[i] and graph[v][i]:
            dfs(i)

def bfs():
    global q, visited
    while q:
        cur = q.pop(0)
        print(cur, end=' ')
        for i in range(1, n+1):
            if not visited[i] and graph[cur][i]:
                visited[i] = 1
                q.append(i)


dfs(v)
print()


visited = [0]*(n+1)
q = [v]
visited[v] = 1
bfs()
```
