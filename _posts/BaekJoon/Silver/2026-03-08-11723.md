## 2. 문제 분석
#### 🐾 생각 흐름

```python
import sys

def input() :
    return sys.stdin.readline().strip()

m = int(input())
s = set()

def add(x) :
    if x not in s :
        s.add(x)
def remove(x) :
    s.discard(x)
def check(x) :
    if x in s :
        return 1
    else :
        return 0
def toggle(x) :
    if x in s :
        s.remove(x)
    else :
        s.add(x)
def all() :
    global s
    s = set(range(1,21))
    return s
def empty() :
    s = set()
    return s
    

for i in range(m) :
    x = list(input().split())
    if x[0] == "add" :
        add(x[1])
    elif x[0] == "check" :
        print(check(x[1]))
    elif x[0] == "remove" :
        remove(x[1])
    elif x[0] == "toggle" :
        toggle(x[1])
    elif x[0] == "all" :
        all()
    elif x[0] == "empty" :
        empty()
    
```
empty()에 global s를 빼먹어서 틀린 줄 알았으나, 이렇게 해도 계속 출력이 잘못나왔다. 


### 🐾 다시 풀이
```python
import sys

def input():
    return sys.stdin.readline().strip()

m = int(input())
s = set()

def add(x):
    s.add(x)

def remove(x):
    s.discard(x)

def check(x):
    if x in s:
        return 1
    else:
        return 0

def toggle(x):
    if x in s:
        s.remove(x)
    else:
        s.add(x)

def all():
    global s
    s = set(range(1, 21))

def empty():
    global s
    s = set()

for _ in range(m):
    x = input().split()
    if x[0] == "add":
        add(int(x[1]))
    elif x[0] == "remove":
        remove(int(x[1]))
    elif x[0] == "check":
        print(check(int(x[1])))
    elif x[0] == "toggle":
        toggle(int(x[1]))
    elif x[0] == "all":
        all()
    elif x[0] == "empty":
        empty()
    
```
[기억할 사항]
x = list(input().split())로 받으면 입력이 예를 들어 "add 3"이라면 x = ["add", "3"]가 저장된다. <br>
<span style="color:red;"><u>이때, split()은 전부 문자열로 나눠서 리스트에 넣기 때문에 숫자여야하는 3이 문자열로써 set에 들어간다 </u></span> <br> 
<span style="color:red;"><u>s = set(range(1,21))는 정수로 숫자를 set에 넣게 된다. </u></span> <br> 
<span style="color:red;"><u>이렇게 자료형이 섞여서 에러가 일어난 것 같다</u></span> <br>

### 🐾 추가적인 생각
질문 게시판을 보니 **비트마스킹**이라는 것으로 풀면 더 빨라진다고 한다. 
```python
import sys
input = sys.stdin.readline

m = int(input())
s = 0

for _ in range(m):
    cmd = input().split()

    if cmd[0] == "add":
        x = int(cmd[1])
        s |= (1 << x)

    elif cmd[0] == "remove":
        x = int(cmd[1])
        s &= ~(1 << x)

    elif cmd[0] == "check":
        x = int(cmd[1])
        print(1 if s & (1 << x) else 0)

    elif cmd[0] == "toggle":
        x = int(cmd[1])
        s ^= (1 << x)

    elif cmd[0] == "all":
        s = (1 << 21) - 2

    elif cmd[0] == "empty":
        s = 0
```
- add x가 s |= (1 << x)로 대체 되었는데, x 번째 비트를 1로 켠다는 의미이다. 
- remove x는 s &= ~(1 << x) 로, x 번째 비트를 0으로 끈다는 의미다. 
- check x는 s & (1 << x), x번째 비트가 1이면 존재한다는 의미
- toggle x는 s ^= (1 << x)로 대체되었는데 0,1 뒤집는 역할
- all 은 s = (1 << 21) - 2로 1~20을 전부 1로 만든다. 
- empty는 s = 0

이는 set보다 메모리가 적고 연산이 전부 비트 연산이기에 O(1)인 풀이라고 한다.
