## Linear Algebra
딥러닝에서 벡터, 행렬, 내적, 고유값 등의 개념이 많이 등장하기 때문에 선형대수를 모델 계산을 이해하는데 필수적이다. <br> <br>

### 📍 NN의 선형구조
입력 벡터  <br>

$\mathbf{x} = (x_1, x_2, \dots, x_d)^T$

가중치 벡터  <br>

$\mathbf{w} = (w_1, w_2, \dots, w_d)^T$

선형 결합  <br>

$z = \mathbf{w}^T \mathbf{x} + b$

출력  <br>

$y = f(z)$

<br><br>
신경망은 이렇게 <span style="background-color: #fff3cd"> 선형 변환 + 비선형 함수</span> 의 반복이다.

### 📍 스칼라와 벡터
- Scalar : 하나의 숫자 (정수 실수 유리수..)
- Vector : 1차원 숫자 배열 (실수 이진수 등..일 수 있음)
- $ \mathbb{R}^n $: n차원 실수 공간

### 📍 내적 (Inner Product (Dot Product))
두 벡터의 유사도(코사인 유사도)를 나타내고자할 때 쓰이는 연산이다. <br>

$x = [x_1, x_2, \dots, x_n]^T,\quad y = [y_1, y_2, \dots, y_n]^T$

$x \cdot y = \|x\| \|y\| \cos \theta$

$= x_1y_1 + x_2y_2 + \dots + x_ny_n = x^T y$

[성질]
- 대칭성: $x\cdot y = y\cdot x$
- 선형성: $\lambda x \cdot y = \lambda (x \cdot y)$
- 양의 정부호성: $x\cdot x = x^T x > 0 \quad (x \ne 0)$
- 분배법칙 : $(x+y)\cdot z = x \cdot z + y \cdot z$

### 📍 Vector Norm
노름은 벡터의 크기를 나타낸다. <br>
노름은 양수성(f(x) = 0이면 x=0), 삼각 부등식(f(x+y) <= f(x) + f(y)>), 동차성(a가 실수라면 f(ax) = |a|f(x))을 만족하는 함수이다. 

- L1 norm : $\|x\|_1 = \sum \lvert x_i \rvert$
- L2 norm (유클리드) : $\|x\|_2 = \sqrt{\sum x_i^2}$
- Infinity norm : $\|x\|_\infty = \max \lvert x_i \rvert$
- Frobenius Norm : $\|A\|_F = \sqrt{Tr(AA^T)}$

<br> 
딥러닝에서는 주로 L2 norm이 쓰인다. ($L^p norm에서 p=2인 경우)

![dl_chp2_11p](/assets/images/dl_chp2_11p.png)


### 📍 Linear Dependency
벡터가 다른 벡터들의 선형결합으로 표현될 수 있으면 **선형종속**이다. <br>
$a_1 x_1 + \cdots + a_k x_k$

<span style="background-color: #fff3cd">선형독립 벡터 개수가 곧 차원이다.</span> <br> <br>

공간을 표현하는 **선형 독립적인 벡터 집합**은 **Basis(기저)**라고 한다. <br>
기저는 선형 공간의 모든 벡터를 유일하게 표현하고, 기저 벡터의 수는 그 공간의 차원과 같다.

$e_1=(1,0,0), e_2=(0,1,0), e_3=(0,0,1)$

- Example: $[5,2,3]^T = 5e_1 + 2e_2 + 3e_3$
- Projection of a vector: $x \cdot e_i = x^T e_i$ <br> $x = [x^T e_1, x^T e_2, \dots, x^T e_n]$
- Orthogonal: $e_i^T e_j = 0,\ \forall i \ne j$
- Normalized: $e_i^T e_i = 1$
If Orthogonal & Normalized = Orthonormal


### 📍 행렬
0차원 = 스칼라
1차원 = 벡터
2차원 = 행렬
더 많은 차원이 있을 수 있고, 숫자들의 배열을 " Tensor"라고함<br>
합연산, 곱연산이 이루어질 수 있는데, 결합 법칙, 분배 법칙이 성립하고, 곱연산 같은 경우에는 교환법칙은 성립하지 않는다. <br>
<br>

전치(Transpose) : $(AB)^T = B^T A^T$ <br>
단위행렬 : $I_n x = x$ <br>
역행렬 : $A^{-1}A = I$ 조건 : square matrix / full rank <br>
<br>

Rank : 행렬에서 선형 독립인 행 또는 열의 개수 $rank(X) \le \min(m,n)$ <br>
정사각행렬에서 rank = n이면 full rank이며, full rank인 정사각 행렬은 비특이행렬, 아니면 특이 행렬이라고 한다. <br> <br>
Trace : 행렬에서 대각선 원소의 합 $Tr(A) = \sum_i A_{ii}$
<br> <br>

### 📍고유값 분해
- 고유값 (Eigenvalue Problem)

$Av = \lambda v$ <br>
$A$: $m \times m$ matrix  <br>
$v$: $m \times 1$ non-zero vector  <br>
$\lambda$: scalar

- 대각화 가능한 행렬의 Eigen Decomposition
$A$는 대각화 가능한 정방 행렬<br>
$V$는 $A$의 고유벡터들로 구성된 행렬<br>
$\Lambda$는 $A$의 고유값들로 구성된 대각 행렬이라고할 때 아래 식과 같다. <br>

$A = V \, \mathrm{diag}(\lambda) \, V^{-1}$
<br>
$\Lambda =
\begin{bmatrix}
\lambda_1 & 0 & \cdots & 0 \\
0 & \lambda_2 & \cdots & 0 \\
\vdots & \vdots & \ddots & \vdots \\
0 & 0 & \cdots & \lambda_n
\end{bmatrix}$
모든 실수 대칭 행렬 $A$는 다음과 같은 고유분해를 가진다.

$A = Q \Lambda Q^T$

$Av = \lambda v$

$\Rightarrow Av - \lambda v = 0$

$\Rightarrow (A - \lambda I)v = 0$

이 방정식에서 $0$이 아닌 $v$ 벡터를 가지려면

$\det(A - \lambda I) = 0$

- 정방 행렬(square matrix)에 대해서만 존재
- 고유값 문제는 정방 행렬에 대해서만 정의됨
- 모든 정방 행렬이 고유값과 고유벡터를 갖는 것은 아님
- 고유벡터의 최대 개수 $m \times m$ 행렬 → 최대 $m$개의 고유벡터
- 행렬이 대각화 가능하면 고유벡터는 서로 독립
- 모든 대칭 행렬의 고유벡터는 서로 직교
- 일반적으로 고유값은 내림차순 정렬
- 행렬 $A$는 고유벡터 방향에 따라 벡터를 고유값만큼 스케일링한다.


### 📍 특이값 분해
$A = U \Sigma V^T$ <br>
$A$: $m \times n$ matrix<br>
$U$: $m \times m$ 직교 행렬 (Left Singular Vectors)<br>
$V$: $n \times n$ 직교 행렬 (Right Singular Vectors)<br>
$\Sigma$: $m \times n$ 대각 행렬 (Singular values)<br><br>

$U^TU = I$, $V^TV = I$<br><br>

## Probability
딥러닝에서 확률은 데이터 분포를 모델링하거나 추론을 수행할 때 등, 다방면에서 쓰인다. 

- 빈도론적 확률 : 사건이 발생하는 비율 <br>
예 : 포커 게임에서 특정 핸드를 뽑을 확률
- 베이지안 확률 : 확신의 질적 수준 (= 신념의 정도) <br>
예 : 의사가 환자를 분석한 후 그가 독감에 걸릴 확률이 40%라고 말하는 경우

### 📍 확률밀도함수(PDF)

연속 확률 변수의 분포 <br>
<br>
- 비음수성(Non-negativity):  확률밀도함수 f(x)>= 0 
- 정규화(Normalization): 확률 밀도 함수의 전체 면적은 1 (=모든 값에 대해 확률 밀도의 적분이 1)
- 확률계산(Probability Calculation): 특정구간[a,b]내에서 연속변수의 확률은 확률밀도함수의 적분으로, 이산변수라면 합산으로
- 값의 해석(Density Interpretation): 확률 밀도 함수의 값 자체는 확률이 아니라 **밀도**


### 📍 부분 확률 분포 (주변확률)
다른 변수는 제거한 확률 <br>
이산 : $P(x) = \sum_y P(x,y)$ <br>
연속 : $p(x) = \int p(x,y)dy$

### 📍 조건부확률
어떤 사건이 주어졌을 때 다른 사건의 확률 <br>
$P(A|B) = \frac{P(A,B)}{P(B)}$

### 📍 독립성
독립이면<br>
$P(x,y) = P(x)P(y)$ 가 성립한다. 
<br>
조건부 독립이면 <br>
$P(x,y|z) = P(x|z)P(y|z)$

### 📍 expectation
확률 변수의 평균값 <br>
이산 : $E[f(x)] = \sum P(x)f(x)$ <br>
연속 : $E[f(x)] = \int p(x)f(x)dx$ <br>
선형성 : $E[af(x)+bg(x)] = aE[f(x)] + bE[g(x)]$

### 📍 Variance
데이터가 평균에서 얼마나 퍼져있는지 (분산) <br>
$Var$(X) = E[(X-E[X])^2]$

### 📍 Covariance (공분산)
두 변수의 선형 관계 $Cov(X,Y) = E[(X-E[X])(Y-E[Y])]$<br><br>
Cov > 0 → 양의 상관  <br>
Cov < 0 → 음의 상관(독립은 아님)  <br>
Cov = 0 → 선형 관계 없음<br>

### 📍 Bernoulli Distribution
(이진 확률 변수) <br>
$P(x=1) = \phi$,$P(x=0) = 1-\phi$

평균 $: E[x] = \phi$

분산 : $Var(x) = \phi(1-\phi)$

### 📍 Gaussian Distribution
(정규분포) <br>
$N(x|\mu,\sigma^2)=\frac{1}{\sqrt{2\pi\sigma^2}}\exp\left(-\frac{(x-\mu)^2}{2\sigma^2}\right)$ <br> <br>

precision : $\beta = \frac{1}{\sigma^2}$

### 📍 시그모이드 함수, 베이즈 정리

$\sigma(x) = \frac{1}{1+e^{-x}}$ <br>

주로 **Bernoulli 확률 모델링**에 사용 <br> <br>
베이즈 정리 : $P(x|y)=\frac{P(y|x)P(x)}{P(y)}$

### 📍 Information Theory
- Self Information : $I(x) = -\log P(x)$, 확률이 낮을수록 정보량(사건이 얼마나 놀라운지)이 큼
- 샤논 엔트로피 : $H(X) = -\sum p(x)\log p(x)$, 불확실성 측정
- KL 분포 : $D_{KL}(P||Q) = \sum P(x)\log\frac{P(x)}{Q(x)}$ 두 분포의 차이 
- 크로스 엔트로피 : $H(P,Q) = -\sum P(x)\log Q(x)$, $H(P,Q) = H(P) + D_{KL}(P||Q)$
- 소프트맥스 : $p_i = \frac{e^{z_i}}{\sum_j e^{z_j}}$
