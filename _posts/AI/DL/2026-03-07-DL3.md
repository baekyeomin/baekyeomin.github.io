## Deep feedfoward network (=MLP=Linear Layers)
input layer -> hidden layer -> output layer <br>
입력이 들어가서 앞으로만 쭉 전달되며 계산됨 = feedfoward <br>
반대로 feed-back연결이 있어서 이전 출력이 다시 입력으로 있다면 = RNN (Recurrent NN)

### 📍 경사하강법
학습데이터$(x^1, y^1), (x^2, y^2), \dots, (x^N, y^N)$가 있을 때, 아래 조건을 최소화하는 최적으 w를 찾는 것이 목표이다. <br>
$E = \frac{1}{2}\sum_n (y^n - \hat{y}^n)^2$ where $\hat{y}^n = f(x^n; w)$ <br> <br>

- 오류 역전파 (Error Backpropagation) : 모델의 파라미터를 반복적으로 업데이트하여 $E$를 감소시키는 방법 $w_i(t+1) = w_i(t) - \epsilon \frac{\partial E}{\partial w_i}$

<span style="color: red">연쇄 법칙 (chain rule) : f(g(x)) 합성함수의미분 : f'(g(x))g'(x) </span>
<br> <br>

- 기울기 계산 

### 📍 퍼셉트론 훈련
모델은 다음과 같은 순서로 학습한다. <br>
1. 현재 파라미터로 예측
2. 예측과 정답의 차이 계산
3. 그 차이를 줄이는 방향으로 gradient 계산
4. 파라미터 업데이트
5. 반복
   <br>
Forward → Loss → Gradient → Update → 반복<br><br>
**경사하강법 문제점** <br>
- 최적화를 위해 일반적으로 반복을 많이하기에 훈련데이터의 크기가 크면 많은 시간이 소요된다. 


## Single Layer Perceptron
장점 : 단일 계층이기 때문에 추론이 빠르고 네트워크 최적화가 용이하다 <br>
단점 : 선형 분류기는 비선형 분류기보다 성능이 낮고 feature을 학습하지 않으며, 훈련데이터의 크기가 클 때 각 에포크에서 기울기를 계산하는데 시간이 많이 소요된다. 

## Multi layer perceptron (MLP)
장점 : 비선형 decision boundary 형성 가능 -> 정확도 향상됨 / 데이터의 더 복잡한 패턴도 학습 가능해져서 일반화 성능이 좋다. <br>
-> Layer 쌍 연결 방법은 CNN (적은 연결), Fully Connectes(완전연결) 등이 있다. <br>


## Deep feedfoward network
딥러닝 전략은 비선형 입출력 매핑을 학습하는 것! <br>
[DNN의 세 가지 문제들]
1. DNN은 최적화를 위해 일반적으로 epoch가 많이 소요된다. (각 반복에서 기울기 계산에 소요되는 시간이 김) <br>
<br>
2. 과적합 -> 학습한 함수가 overfitting 가능<br>
3. 기울기 소실 문제 -> 하층의 기울기가 일반적으로 매우 작음 (다층 신경망 최적화 하는데 매우 많은 시간 소요)
<br><br>

[DNN 훈련]
1. 네트워크를 설계하고 데이터를 준비한다
2. 네트워크를 초기화한다
3. 미니배치 SGD 루프
- 데이터 배치를 샘플링한다
- 배치를 네트워크르 통해 순방향 전파하여 손실을 계산한다. 
- 역전파를 통해 기울기를 계산한다. 
- 기울기를 사용하여 파라미터를 업데이트한다.
